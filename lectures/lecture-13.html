Lecture 13

Before Lecture

Before lecture 13, please watch the following videos from Module 8: From Examples 1, and From Examples 2. Also be sure to answer the associated online questions.

During Lecture

The plan for lecture is:

  • Design an abstract fn from examples, and rewrite the original functions to call it. This particular problem will involve a little more complexity than the pre-lecture examples. Develop the signature for abstract functions using type inference. Design an abstract function based on a template. Write entirely new functions that use abstract functions in their function definition. (Some of this will be homework.) For extra practice using built-in abstract functions: m08-raining-eggs-refactor-starter.rkt

    • m01-functions-starter.rkt
