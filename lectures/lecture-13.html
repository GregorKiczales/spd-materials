Lecture 13

Before Lecture

Before lecture 13, please watch the following videos from Module 8: From Examples 1, and From Examples 2. Also be sure to answer the associated online questions.

During Lecture

The starters for this lecture are:

  • m08-from-examples-starter.rkt
  • m08-use-bia-fn-starter.rkt

The plan for lecture is:

  • Design an abstract fn from examples, and rewrite the original functions to call it. The lecture problem will involve more complexity than the pre-lecture examples.
  • Develop the signature for abstract functions using type inference.
  • Learn how to identify which build-in abstract functions can help with a given function body.
